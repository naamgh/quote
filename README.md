# Email Response Builder

GitHub-friendly file layout:

- `index.html` — app shell
- `styles.css` — visual styles
- `app.js` — app logic
- `email-elements.json` — saved email elements for Tabs 1–6
- `quote-items.json` — quote builder price list

## Updating saved email elements

Use **Export elements** in the Saved elements column to download a fresh `email-elements-YYYY-MM-DD.json` file. Rename it to `email-elements.json` and replace the file in this folder/GitHub repository.

Use **Import & merge** or **Import & replace** to restore a backup from JSON.

## Updating quote items

Use the existing quote price list export/import buttons. To update the hosted default list, replace `quote-items.json` in GitHub.
